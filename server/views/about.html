How to play:

To play the game, you must first register an account. Once you have registered, you can log in and join a game session.
The game will start once the host has started the game session. You will be presented with a series of questions, and you must answer them within the time limit. The faster you answer, the more points you will earn.
The game will end once all questions have been answered. The player with the most points wins!

Description:

Online quiz game where users can compete in real-time by answering questions across various categories.

Functionalities:

  • Players can register, log in, and join games (to answer questions within a time limit and earn points based on correct answers)
  • Questions will be retrieved from an API called 'Open Trivia Database' or added by users (possibly with an admin role)
  • Scores will be stored in a database to display a leaderboard featuring the top players with statistics about their games

    • Technology stack:

    This project will be built using the following technologies:
  • Frontend: HTML, CSS, JavaScript for creating an interactive user interface
  • Backend: NodeJS + Express.js
  • Database: MongoDB / MySQL for storing user data, questions, and game-related information
